# DevConnect - Developer Job Board


A full-stack MERN application where developers 
can find jobs and recruiters can post them.


## Live Demo 
- Frontend: https://devconnect.vercel.app
- Backend: https://devconnect-api.onrender.com

## Features 
- JWT Authentication 
- Developer & Recruiter Profiles
- Job Posting & Search 
- Apply for Jobs
- Email Notifications
- Admin Panel 
- Image Upload (Cloudinary)

## Tech Stack 
- MongoDB, Express, React, Node.js
- Tailwind CSS, JWT, Cloudinary, Nodemailer

## Setup
### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm run dev
